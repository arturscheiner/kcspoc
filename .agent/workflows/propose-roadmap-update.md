# Propose Roadmap Update

Purpose:
Help propose a structured update to the roadmap without applying it.

Steps:
1. Identify which roadmap file would be affected.
2. Explain why the update is needed.
3. Describe the impact on existing roadmap items.
4. Highlight risks and trade-offs.

Important:
- This workflow MUST NOT modify any roadmap file.
- It produces a proposal for human review only.

Output format:
- Target roadmap file
- Proposed change summary
- Rationale
- Risks
