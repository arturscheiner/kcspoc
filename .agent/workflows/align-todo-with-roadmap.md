# Align TODO with Roadmap

Purpose:
Validate whether TODO.md items align with the active roadmap.

Steps:
1. Scan TODO.md.
2. Identify which items belong to:
   - current stabilization,
   - future development,
   - or should be moved out.
3. Flag items that do not align with the roadmap.
4. Suggest cleanup or reclassification.

Restrictions:
- Do NOT edit TODO.md automatically.
- Provide suggestions only.

Output format:
- Aligned items
- Misaligned items
- Suggested actions
