This workspace contains the kcspoc project.

kcspoc is a Bash-based CLI tool designed for Pre-Sales Engineers to streamline Proof of Concept deployments of Kaspersky Container Security.

Project rules:
- Target audience: Pre-Sales Engineers and technical consultants.
- Execution environment: enterprise Linux systems.
- Primary goals: reliability, clarity, operational safety.
- This project is written entirely in Bash.
- This project uses Git tags and GitHub Releases for stable versions.
- The main branch is not assumed to be stable.
